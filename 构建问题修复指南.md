# 构建问题修复指南

## 问题描述

构建时出现错误：`"default" is not exported by "src/app/App.tsx"`

## 原因

在优化过程中，App.tsx 文件可能被意外修改或清空。

## 解决方案

### 方案1：恢复 App.tsx 文件（推荐）

如果您有备份或 Git 版本控制，请恢复原始的 App.tsx 文件。

### 方案2：确保 App.tsx 有默认导出

在 App.tsx 文件末尾确保有：

```typescript
export default function App() {
  // ... 组件代码
}
```

或者：

```typescript
function App() {
  // ... 组件代码
}

export default App;
```

### 方案3：添加性能监控导入

在 App.tsx 中添加以下导入和初始化：

```typescript
import { initPerformanceMonitoring } from './utils/performance';

// 在 useEffect 中
useEffect(() => {
  // ... 其他初始化代码
  
  // 初始化性能监控
  if (import.meta.env.DEV) {
    initPerformanceMonitoring();
  }
}, []);
```
