# 快速修复指南 - 模块导入错误

## 🔧 问题描述

**错误信息**:
```
Failed to fetch dynamically imported module: 
http://localhost:5173/src/app/pages/StoryboardEditor/index.tsx
```

**原因**: 缺少新增的依赖包

---

## ✅ 解决方案

### 步骤 1: 安装缺失的依赖

```bash
cd "d:/桌面/剧本改20"
npm install zod @tanstack/react-virtual
```

### 步骤 2: 重启开发服务器

```bash
# 停止当前服务器 (Ctrl+C)
# 然后重新启动
npm run dev
```

### 步骤 3: 清除浏览器缓存

- 按 `Ctrl + Shift + R` 强制刷新
- 或者清除浏览器缓存后刷新

## ⚠️ 如果问题仍然存在

### 方案 1: 完全重新安装
```bash
rm -rf node_modules
rm package-lock.json
npm install
npm run dev
```

### 方案 2: 检查导入路径
确保所有新文件的导入路径正确：

```typescript
// ✅ 正确
import { z } from 'zod';
import { useVirtualizer } from '@tanstack/react-virtual';

// ❌ 错误
import { z } from 'zod/lib';
import { useVirtualizer } from '@tanstack/virtual';
```

### 方案 3: 检查 TypeScript 配置
确保 `tsconfig.json` 包含：
```json
{
  "compilerOptions": {
    "moduleResolution": "bundler",
    "resolveJsonModule": true
  }
}
```
