## 🚀 使用指南

### 快速开始

1. **导入功能**
```typescript
import { useBatchSelection } from './hooks/useBatchSelection';
import { useVersionHistory } from './hooks/useVersionHistory';
import { useAutoBackup } from './hooks/useAutoBackup';
```

2. **初始化 Hook**
```typescript
const batchSelection = useBatchSelection(items.length);
const versionHistory = useVersionHistory(projectId);
const autoBackup = useAutoBackup(projectId, assets);
```

3. **渲染组件**
```typescript
<BatchActionsBar {...batchSelection} />
<VersionHistoryDialog {...versionHistory} />
```

### 常见问题

**Q: 虚拟滚动不流畅？**  
A: 检查 itemHeight 是否准确，使用 React.memo 优化组件

**Q: 备份失败？**  
A: 检查 localStorage 是否已满，清理旧备份

**Q: 图片加载慢？**  
A: 使用渐进式加载，提供低质量占位图
