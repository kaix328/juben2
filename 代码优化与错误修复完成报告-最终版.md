## 🐛 修复的错误

### 1. 数据迁移错误
**问题：** `useAutoMigration.ts:55` - `Cannot read properties of undefined (reading 'map')`

**原因：** `validateStoryboard` 函数在访问 `storyboard.panels.map()` 前没有检查对象是否存在

**解决方案：**
```typescript
// 在 validateStoryboard 函数中添加安全检查
if (!storyboard || typeof storyboard !== 'object') {
    return { success: false, errors: [...] };
}

if (!sb.panels) {
    sb.panels = [];
}

if (!Array.isArray(sb.panels)) {
    return { success: false, errors: [...] };
}
```

**修复文件：**
- `src/app/utils/validation/storyboardValidation.ts`
- `src/app/hooks/useAutoMigration.ts`

---

### 2. React Query undefined 错误
**问题：** `Query data cannot be undefined` 警告出现在多个查询中

**原因：** React Query 不允许查询函数返回 `undefined`，但某些存储函数可能返回 `undefined`

**解决方案：** 使用空值合并运算符 `??` 确保返回 `null` 或 `[]` 而不是 `undefined`

**修复的查询函数：**

#### useAssets.ts
```typescript
// 修复前
return await assetStorage.getByProjectId(projectId);

// 修复后
const assets = await assetStorage.getByProjectId(projectId);
return assets ?? null;
```

#### useProject.ts
```typescript
// useProject
const project = await projectStorage.getById(projectId);
return project ?? null;

// useProjects
const projects = await projectStorage.getAll();
return projects ?? [];

// useProjectByChapter
const project = await projectStorage.getById(projectId);
return project ?? null;
```

#### useStoryboard.ts
```typescript
// useStoryboards
const storyboards = await storyboardStorage.getAll();
return storyboards ?? [];

// usePrefetchStoryboard
const storyboard = await storyboardStorage.getByChapterId(chapterId);
return storyboard ?? null;
```

**修复文件：**
- `src/app/hooks/queries/useAssets.ts`
- `src/app/hooks/queries/useProject.ts`
- `src/app/hooks/queries/useStoryboard.ts`
