## 💡 使用指南

### 1. 质量检查工作流

```
1. 点击"质量检查"按钮
   ↓
2. 查看质量报告（分数、问题分类）
   ↓
3. 筛选和查看具体问题
   ↓
4. 点击问题跳转到对应分镜
   ↓
5. 选择问题进行一键优化
   ↓
6. 查看优化结果
```

### 2. 连贯性检查工作流

```
1. 点击"连贯性检查"按钮
   ↓
2. 查看连贯性报告
   ↓
3. 查看专业规则检查结果
   ↓
4. 点击问题跳转到对应分镜
   ↓
5. 根据建议手动调整
```

### 3. 提示词优化工作流

```
1. 在分镜卡片中点击"提示词预览"
   ↓
2. 查看图像/视频提示词
   ↓
3. 选择目标平台
   ↓
4. 查看质量分析和建议
   ↓
5. 编辑并保存提示词
```

### 4. 分镜排序工作流

```
1. 切换到列表视图
   ↓
2. 直接拖拽分镜卡片
   ↓
3. 自动重新编号和保存
```

```typescript

  variant="ghost"
  size="sm"

  title="预览提示词"
>
  <Eye className="w-4 h-4" />
  提示词
</Button>
```

### 2. 数据分析功能

数据分析需要项目 ID，确保在调用时传递：

```typescript

```
