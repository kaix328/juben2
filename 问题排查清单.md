### 10. 常见问题及解决方案

#### 问题1：按钮点击无反应
**可能原因：**
- onClick 事件未正确绑定
- 函数作用域问题
- 事件被阻止传播

**解决方法：**
```typescript
// 确保 onClick 正确绑定
<Button onClick={() => handleAIGenerate('fullBody')}>
  AI生成
</Button>

// 而不是
<Button onClick={handleAIGenerate('fullBody')}>  // ❌ 错误
```

#### 问题2：提示词未生成
**可能原因：**
- PromptEngine 初始化失败
- character 对象为空
- onUpdate 函数未正确传递

**解决方法：**
```typescript
// 添加调试日志
const handleAIGenerate = (type: 'fullBody' | 'face') => {
  console.log('handleAIGenerate 被调用', type);
  console.log('character:', character);
  console.log('project:', project);
  
  // ... 原有代码
};
```

#### 问题3：toast 提示不显示
**可能原因：**
- sonner 未正确安装
- Toaster 组件未添加到根组件

**解决方法：**
```bash
# 检查 sonner 是否安装
npm list sonner

# 如果未安装
npm install sonner
```

```typescript
// 确保在 App.tsx 或根组件中添加了 Toaster
import { Toaster } from 'sonner';

function App() {
  return (
    <>
      <Toaster />
      {/* 其他组件 */}
    </>
  );
}
```
