## 🔄 迁移流程

### 自动迁移（用户无感知）
```
1. 用户打开分镜页面
   ↓
2. useStoryboard 加载数据
   ↓
3. autoMigrateStoryboard 检测版本
   ↓
4. 如果是旧版本，自动迁移
   ↓
5. 验证迁移后的数据
   ↓
6. 自动保存到存储
   ↓
7. 返回迁移后的数据给 UI
```

### 手动批量迁移（可选）
```typescript
// 在设置页面提供按钮
import { batchMigrateAllStoryboards } from '@/hooks/useAutoMigration';

const handleMigrate = async () => {
    const result = await batchMigrateAllStoryboards();
    console.log(`迁移完成: ${result.migrated}/${result.total}`);
};
```
