## 💡 使用提示

### 批量操作
1. 点击"批量操作"按钮
2. 选择多个角色（点击卡片或复选框）
3. 底部会出现浮动工具栏
4. 选择操作（添加标签、删除、生成图片）

### 高级搜索
1. 点击"高级搜索"按钮
2. 设置筛选条件（关键词、标签、图片状态等）
3. 点击"搜索"
4. 查看筛选结果

### 版本历史
1. 选择一个角色
2. 修改角色信息
3. 点击"版本历史"按钮
4. 查看历史版本
5. 点击"回滚"恢复到旧版本

### 模板系统
1. 点击"模板"按钮
2. 浏览内置模板或自定义模板
3. 选择模板，点击"应用"
4. 新角色会自动创建并填充模板数据

### 自动备份
- 系统每5分钟自动备份
- 点击"备份管理"查看所有备份
- 可以手动创建、恢复、导出备份
