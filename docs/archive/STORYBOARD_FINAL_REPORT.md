## 🎯 新增的功能详解

### 1. AI 质量检查系统 ⭐⭐⭐⭐⭐

**位置**: 顶部工具栏 "质量检查" 按钮（Sparkles 图标）

**功能**:
- 🎯 自动检测 6 大类问题
  - 连贯性问题（轴线、景别跳跃等）
  - 时长问题（过长、过短）
  - 角色问题（角色追踪）
  - 镜头问题（缺少建立镜头）
  - 对话问题（对话长度）
  - 逻辑问题（编号、ID重复）
- 📊 质量分数评级（0-100分）
- 📈 数据分析和趋势图表
- 🔍 问题筛选和搜索
- 💡 智能优化建议
- 🎯 点击问题跳转到对应分镜

**使用方式**:
1. 点击顶部 "质量检查" 按钮
2. 右侧显示质量报告侧边栏
3. 查看质量分数和问题分类
4. 点击问题跳转到对应分镜
5. 选择问题进行一键优化

---

### 2. 连贯性检查对话框 ⭐⭐⭐⭐⭐

**位置**: 顶部工具栏 "连贯性检查" 按钮（CheckCircle2 图标）

**功能**:
- 🎬 专业电影制作规则检查
- 📐 轴线规则检测（180度规则）
- 🎥 30度规则检测
- 📹 景别跳跃检查
- 🔄 镜头连贯性分析
- 💡 专业修复建议

**使用方式**:
1. 点击顶部 "连贯性检查" 按钮
2. 弹出连贯性报告对话框
3. 查看问题统计（错误/警告/提示）
4. 点击问题跳转到对应分镜
5. 根据建议手动调整

---

### 3. 一键优化功能 ⭐⭐⭐⭐

**位置**: 质量报告侧边栏中的问题列表

**功能**:
- 🤖 AI 自动修复分镜问题
- ✅ 批量选择问题优化
- 📊 优化结果统计
- ⚡ 自动更新分镜数据

**优化类型**:
- 时长问题：自动调整到合理范围（3-10秒）
- 镜头问题：根据建议调整参数
- 对话/角色问题：重新生成优化的提示词

**使用方式**:
1. 在质量报告中查看问题列表
2. 点击 "一键优化" 按钮
3. 选择要优化的问题
4. 点击 "优化选中的 X 个问题"
5. 查看优化结果

---

### 4. 提示词预览对话框 ⭐⭐⭐⭐⭐

**位置**: 通过 Context 调用（已集成）

**功能**:
- 🎨 图像和视频双模式
- 🌐 支持 8+ AI 平台
  - Midjourney
  - Stable Diffusion
  - nanobananaPro
  - Runway
  - Pika
  - Sora
  - Kling (快手)
- 📝 提示词质量分析
- 🔄 原始/新生成对比模式
- 💡 优化建议系统
- 🎯 专业增强版 v2.0
- 📋 一键复制到剪贴板

**使用方式**:
```typescript
// 通过 Context 调用
handleOpenPromptPreview(panel)
```

**注意**: 如需在 ShotCard 中添加快捷按钮，可以在操作按钮区域添加：
```typescript
<Button onClick={() => handleOpenPromptPreview?.(panel)}>
  <Eye className="w-4 h-4" />
  预览提示词
</Button>
```
