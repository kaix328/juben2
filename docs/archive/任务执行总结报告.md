## 🔄 未完成的任务

### 1. 修复 ShotCard 测试失败（31个）

**问题**: `Cannot read properties of undefined (reading 'join')`

**原因**: 代码中多处直接调用数组方法，未检查 undefined

**需要修复的位置**:
```typescript
// src/app/components/storyboard/ShotCard.tsx

// 需要修改的代码模式：
panel.characters.join(', ')  // ❌ 错误
(panel.characters || []).join(', ')  // ✅ 正确

// 涉及的属性：
- panel.characters
- panel.props
- panel.soundEffects
- panel.characterActions
- panel.vfx
```

**手动修复步骤**:
1. 打开 `src/app/components/storyboard/ShotCard.tsx`
2. 搜索 `.join(`
3. 为所有数组访问添加 `|| []` 检查
4. 保存文件
5. 运行 `npm test -- ShotCard` 验证
