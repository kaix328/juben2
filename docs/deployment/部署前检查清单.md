## 📋 部署步骤

### 1. 准备阶段
```bash
# 1. 拉取最新代码
git pull origin main

# 2. 安装依赖
npm install

# 3. 构建生产版本
npm run build

# 4. 测试构建产物
npm run preview
```

### 2. 部署阶段
```bash
# 1. 停止旧服务
pm2 stop app

# 2. 备份旧版本
mv dist dist_backup

# 3. 部署新版本
mv dist_new dist

# 4. 启动新服务
pm2 start app
```

### 3. 验证阶段
```bash
# 1. 检查服务状态
pm2 status

# 2. 查看日志
pm2 logs app

# 3. 访问应用
curl http://localhost:5173/health
```
