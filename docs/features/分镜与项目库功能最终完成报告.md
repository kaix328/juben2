## 🚀 使用指南

### 分镜功能使用

**自动使用**（已集成）:
```
1. 创建或打开项目
2. 生成分镜
3. 查看控制台输出：
   [Prompt Validation] 总体评分: 85/100
   [Quality Check] 质量分数: 88/100
```

**手动优化**:
```typescript
import { quickOptimize } from '@/utils/batchOptimizer';

const result = await quickOptimize(panels, characters, scenes);
setPanels(result.optimizedPanels);
```

### 项目库功能使用

**启用预览**:
```
1. 打开项目库页面
2. 找到工具栏中的 [👁️ 预览] 开关
3. 点击启用
4. 点击任意生成按钮
5. 在预览对话框中查看优化效果
6. 可手动编辑提示词
7. 点击"确认生成"
```

**批量生成**:
```
1. 点击工具栏中的"批量生成"按钮
2. 在对话框中查看待生成列表
3. 点击"开始生成"
4. 可随时暂停/恢复/取消
5. 查看生成进度和结果
```
