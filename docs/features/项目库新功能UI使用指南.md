## 🎯 新增功能位置

### 1. 提示词预览开关 👁️

**位置：** 项目库页面顶部工具栏

**外观：**
```
[👁️ 预览] ← 开关按钮（眼睛图标 + Switch 开关）
```

**功能：**
- 启用后，点击任何"生成"按钮时会先显示预览对话框
- 可以查看优化后的提示词
- 显示质量评分（0-100）
- 可以手动编辑提示词
- 查看优化建议

**使用方法：**
1. 点击"预览"开关启用
2. 点击任意角色/场景的"生成"按钮
3. 会弹出提示词预览对话框
4. 查看并编辑提示词
5. 点击"确认生成"开始生成

---

### 2. 批量生成进度对话框 📊

**触发方式：** 点击"批量生图"按钮

**位置：** 项目库页面顶部工具栏

**外观：**
```
[🔄 批量生图] ← 绿色按钮
```

**功能：**
- 自动批量生成所有缺失的图片
- 显示实时进度条
- 显示每个任务的状态（等待/进行中/成功/失败）
- 可以暂停/继续
- 可以取消
- 显示详细错误信息

**使用方法：**
1. 点击"批量生图"按钮
2. 确认要生成的数量
3. 自动弹出进度对话框
4. 查看实时进度
5. 可以点击"暂停"暂停生成
6. 可以点击"取消"停止生成

- 🟢 可用："批量生图"
- 🔄 生成中："生成中..."（图标旋转）
- ⚫ 禁用：灰色不可点击

## 🎯 使用场景

### 场景 1：精细控制提示词

**需求：** 想要手动调整提示词以获得更好的效果

**步骤：**
1. ✅ 启用"预览"开关
2. ✅ 点击角色的"生成全身图"按钮
3. ✅ 在预览对话框中查看优化后的提示词
4. ✅ 根据质量评分和建议进行调整
5. ✅ 手动编辑提示词
6. ✅ 点击"确认生成"

**效果：** 生成质量更符合预期

---

### 场景 2：快速批量生成

**需求：** 一次性生成所有缺失的图片

**步骤：**
1. ✅ 关闭"预览"开关（避免每次都弹窗）
2. ✅ 点击"批量生图"按钮
3. ✅ 确认生成数量
4. ✅ 查看实时进度
5. ✅ 等待完成或随时暂停

**效果：** 高效完成大量图片生成

---

### 场景 3：监控批量生成进度

**需求：** 了解批量生成的详细情况

**步骤：**
1. ✅ 点击"批量生图"
2. ✅ 在进度对话框中查看：
   - 总进度（3/10）
   - 每个任务的状态
   - 成功/失败统计
   - 失败任务的错误信息
3. ✅ 如果发现问题，点击"暂停"
4. ✅ 检查失败原因
5. ✅ 点击"继续"或"取消"

**效果：** 完全掌控生成过程

## 💡 使用技巧

### 技巧 1：提示词预览的最佳实践

✅ **推荐做法：**
- 首次使用时启用预览，了解优化效果
- 对重要角色启用预览，精细调整
- 批量生成时关闭预览，提高效率

❌ **不推荐：**
- 批量生成时启用预览（会弹出很多对话框）

---

### 技巧 2：批量生成的最佳实践

✅ **推荐做法：**
- 生成前先保存项目
- 网络不稳定时使用暂停功能
- 查看失败任务的错误信息
- 失败后可以单独重新生成

❌ **不推荐：**
- 生成过程中刷新页面（会丢失进度）
- 忽略错误信息直接重试

---

### 技巧 3：提示词质量优化

**查看质量评分：**
- 90-100：优秀 ⭐⭐⭐⭐⭐
- 80-89：良好 ⭐⭐⭐⭐
- 70-79：一般 ⭐⭐⭐
- 60-69：需改进 ⭐⭐
- <60：较差 ⭐

**根据建议优化：**
- 红色问题：必须修复
- 黄色警告：建议优化
- 蓝色建议：可选改进

## 🔧 故障排除

### 问题 1：预览开关不显示

**原因：** 浏览器缓存

**解决：**
1. 按 `Ctrl + Shift + R` 强制刷新
2. 清除浏览器缓存
3. 重启开发服务器

---

### 问题 2：批量生成卡住

**原因：** 网络问题或 API 限制

**解决：**
1. 点击"暂停"
2. 检查网络连接
3. 等待几分钟
4. 点击"继续"
5. 如果仍然失败，点击"取消"并单独重试失败的任务

---

### 问题 3：预览对话框不弹出

**原因：** 预览开关未启用

**解决：**
1. 检查工具栏中的"预览"开关
2. 确保开关为蓝色（启用状态）
3. 重新点击生成按钮
