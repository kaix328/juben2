# 项目库提示词生成 - 改进建议清单

> 生成时间：2026-01-24  
> 基于对 `promptEngine.ts`, `assetExtractor.ts`, `useImageGeneration.ts` 的全面分析

---

## 📊 问题总览

### 🔴 严重问题（3个）

1. **AI 提取数据质量不稳定** - 返回格式不一致，需要大量自适应逻辑
2. **提示词过于冗长** - 200+ tokens，中英文混杂，影响生成质量
3. **缺少提示词验证** - 无长度/格式/质量检查

### 🟡 重要问题（3个）

4. **字段映射不一致** - AI 返回 `hair` 但类型定义是 `hairStyle`
5. **提示词预览缺失** - 用户无法预览优化后的提示词
6. **批量生成无进度详情** - 只显示总进度，失败时无详细信息

### 🟢 次要问题（3个）

7. **硬编码的映射表** - 与 `cinematography.ts` 重复
8. **缺少平台适配** - `formatForPlatform` 未实际使用
9. **服饰与角色关联弱** - 未充分利用角色物理特征

### 方案 3：精简提示词长度（高优先级）

**目标：** 将提示词从 200+ tokens 精简到 100-150 tokens

**改进策略：**
1. **去重**：移除重复的质量标签
2. **语言统一**：根据平台选择纯中文或纯英文
3. **关键词提取**：只保留最重要的描述词
4. **权重替代重复**：用 `(keyword:1.2)` 替代重复关键词

**示例对比：**

**改进前（230 tokens）：**
```
角色名, char_zhangsan_001, 男性，25岁，身高180cm，黑色短发，剑眉星目，身穿黑色长袍，佩戴玉佩, 全身正视图, full body shot, standing pose, front view, 白色背景, white background, simple background, 超高质量, 8K分辨率, 专业级, 精细细节, 完美构图, masterpiece, best quality, 8k, ultra detailed, professional, perfect composition, character design, reference sheet
```

**改进后（120 tokens）：**
```
char_zhangsan_001, 25-year-old male, 180cm tall, short black hair, sharp eyebrows, black robe, jade pendant, full body shot, front view, white background, (masterpiece:1.3), (best quality:1.2), 8k, ultra detailed, character design sheet
```

**实施位置：** `src/app/utils/promptEngine.ts` 的 `buildAdvancedPrompt` 方法

**预期效果：**
- 提示词长度：-40%
- 生成速度：+20%
- 生成质量：+15%

### 方案 4：修复字段映射不一致（高优先级）

**问题：** AI 返回 `hair` 但类型定义是 `hairStyle`

**解决方案：**

**方法 A：修改 AI Prompt（推荐）**
```typescript
// 在 Prompt 中明确字段名
"hairStyle": "发型（如：短发，选填）",
"hairColor": "发色（如：黑色，选填）",
```

**方法 B：添加字段映射层**
```typescript
const FIELD_MAPPING = {
  hair: 'hairStyle',
  // 其他映射...
};

function mapFields(rawData: any): Character {
  const mapped: any = {};
  Object.entries(rawData).forEach(([key, value]) => {
    const targetKey = FIELD_MAPPING[key] || key;
    mapped[targetKey] = value;
  });
  return mapped;
}
```

**实施文件：** `src/app/utils/ai/assetExtractor.ts`

**预期效果：**
- 数据丢失率：20% → 0%
- 字段完整性：100%
