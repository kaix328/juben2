## 💡 使用方式

### 查看关系图谱
1. 切换到 **角色标签页**
2. 点击 **"关系图谱"** 按钮
3. 在对话框中查看所有资产的关系
4. 点击 **"自动分析关系"** 生成关系
5. 点击任意节点跳转到对应资产

### 从其他标签页访问
如果在场景/道具/服装标签页想查看关系图谱：
1. 切换到 **角色标签页**
2. 点击 **"关系图谱"** 按钮
3. 在图谱中可以看到所有类型的资产
4. 点击对应节点可以跳转回去
