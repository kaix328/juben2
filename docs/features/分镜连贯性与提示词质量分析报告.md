## 🔍 发现的问题

### 1. 连贯性问题

#### 问题 1.1：角色触发词不一致
**现状**：
- 视频提示词中显示：`张三(char_zhange09_f456): 30岁男性`
- 但如果角色库中角色没有触发词，会显示为空或错误

**影响**：
- 角色外观在不同分镜中可能不一致
- AI 无法识别同一角色

**已解决** ✅：
- 刚刚优化了触发词生成系统
- 自动为所有角色生成有意义的触发词

#### 问题 1.2：场景环境描述缺失
**现状**：
```typescript
// panelProcessor.ts 中有智能填充
if (!panel.environmentMotion) {
    // 根据场景推断环境动态
}
```

**问题**：
- 如果场景信息不完整，环境描述可能过于简单
- 缺少场景的光线、天气、氛围等细节

**建议**：
- 在生成分镜时，强制要求场景必须包含完整信息
- 从资产库中提取场景的详细描述

#### 问题 1.3：镜头运动连贯性
**现状**：
```typescript
// 已有上下文感知逻辑
const prevMovement = prevPanel?.cameraMovement || '';
const getCoherentMovement = (): string => {
    if (isSceneStart) return '静止';
    if (prevMovement === '聚焦推进') return '静止';
    // ...
}
```

**问题**：
- 逻辑较简单，可能产生不自然的运镜序列
- 缺少对整个场景节奏的把控

**建议**：
- 引入"场景节奏分析"，根据情绪曲线调整运镜
- 添加"运镜禁忌规则"（如：不连续3次推镜）

### 2. 提示词质量问题

#### 问题 2.1：提示词缺少关键信息
**检查代码**：
```typescript
// qualityChecker.ts 已有检查
if (!aiPrompt.trim()) {
    issues.push({
        message: '缺少绘画提示词',
        severity: IssueSeverity.WARNING
    });
}
```

**发现的缺失**：
1. **时间信息缺失** - 提示词中没有明确的时间（白天/夜晚/黄昏）
2. **天气信息缺失** - 没有天气描述（晴天/雨天/雪天）
3. **情绪氛围缺失** - 缺少情绪关键词（紧张/浪漫/悲伤）
4. **构图信息不完整** - 只有景别，缺少具体构图方式

#### 问题 2.2：提示词过于冗长
**现状**：
```typescript
// promptEngine.ts 生成的提示词可能包含：
// - 角色信息（名称+触发词+外貌）
// - 场景信息（地点+环境+天气）
// - 镜头参数（景别+角度+运动）
// - 导演风格（艺术风格+色调+光影）
// - 质量标签（多个）
```

**问题**：
- 提示词可能超过200字，AI模型难以处理
- 信息过多导致重点不突出

**建议**：
- 实现"提示词优先级系统"
- 核心信息（角色+场景+景别）权重最高
- 次要信息（质量标签）可以精简

#### 问题 2.3：中英文混杂
**现状**：
```typescript
// promptEngine.ts 中
parts.push({ type: 'subject', value: `张三(char_zhange09_f456)`, language: 'zh' });
parts.push({ type: 'technical', value: 'medium shot, eye level', language: 'en' });
```

**问题**：
- 中英文混合可能影响某些AI模型的理解
- 不同平台对语言的支持不同

**建议**：
- 根据目标平台选择语言
- 可灵/豆包：纯中文
- Runway/Pika：纯英文或中英混合

### 3. 数据完整性问题

#### 问题 3.1：必填字段可能为空

**建议**：
- 在质量检查中添加"必填字段检查"

#### 问题 3.2：角色信息不完整
**现状**：

**影响**：
- 提示词中角色描述不准确
- 不同分镜中角色外观不一致

**建议**：
- 在资产库中强制要求角色必须有外貌描述

#### 4.1 提示词压缩
```typescript
// 新增：promptCompressor.ts
export function compressPrompt(prompt: string, maxLength: number = 200): string {
    // 1. 移除重复词汇
    const words = prompt.split(/[,，]/);
    const uniqueWords = [...new Set(words)];
    
    // 2. 按优先级排序
    const prioritized = prioritizeWords(uniqueWords);
    
    // 3. 截取到目标长度
    let compressed = '';
    for (const word of prioritized) {
        if ((compressed + word).length > maxLength) break;
        compressed += word + ', ';
    }
    
    return compressed.trim();
}

function prioritizeWords(words: string[]): string[] {
    const priority = {
        character: 10,  // 角色名
        trigger: 9,     // 触发词
        shot: 8,        // 景别
        location: 7,    // 地点
        action: 6,      // 动作
        mood: 5,        // 情绪
        style: 4,       // 风格
        quality: 1      // 质量标签
    };
    
    return words.sort((a, b) => {
        const scoreA = getWordPriority(a, priority);
        const scoreB = getWordPriority(b, priority);
        return scoreB - scoreA;
    });
}
```

```typescript
// src/app/utils/promptValidator.ts
export interface PromptValidationResult {
    valid: boolean;
    score: number;
    issues: string[];
    suggestions: string[];
}

export function validateStoryboardPrompt(
    panel: StoryboardPanel,
    characters: Character[],
    scenes: Scene[]
): PromptValidationResult {
    const issues: string[] = [];
    const suggestions: string[] = [];
    let score = 100;
    
    // 1. 检查提示词是否存在
    if (!panel.aiPrompt || panel.aiPrompt.trim() === '') {
        issues.push('缺少绘画提示词');
        suggestions.push('点击"刷新提示词"按钮生成');
        score -= 30;
    }
    
    // 2. 检查提示词长度
    const promptLength = panel.aiPrompt?.length || 0;
    if (promptLength < 40) {
        issues.push('提示词过短');
        suggestions.push('添加更多场景和氛围描述');
        score -= 10;
    } else if (promptLength > 300) {
        issues.push('提示词过长');
        suggestions.push('精简次要信息，保留核心描述');
        score -= 5;
    }
    
    // 3. 检查角色信息
    if (panel.characters.length > 0) {
        const prompt = panel.aiPrompt?.toLowerCase() || '';
        const missingChars = panel.characters.filter(
            name => !prompt.includes(name.toLowerCase())
        );
        if (missingChars.length > 0) {
            issues.push(`提示词中缺少角色: ${missingChars.join('、')}`);
            suggestions.push('在提示词中添加角色名称和触发词');
            score -= 15;
        }
    }
    
    // 4. 检查场景信息
    const scene = scenes.find(s => s.id === panel.sceneId);
    if (scene && panel.aiPrompt) {
        const prompt = panel.aiPrompt.toLowerCase();
        if (!prompt.includes(scene.location?.toLowerCase() || '')) {
            issues.push('提示词中缺少场景地点');
            suggestions.push(`添加场景地点: ${scene.location}`);
            score -= 10;
        }
    }
    
    // 5. 检查镜头参数
    if (panel.aiPrompt && !panel.aiPrompt.includes(panel.shot)) {
        issues.push('提示词中缺少景别信息');
        suggestions.push(`添加景别: ${panel.shot}`);
        score -= 5;
    }
    
    return {
        valid: issues.length === 0,
        score: Math.max(0, score),
        issues,
        suggestions
    };
}
```

### 示例2：场景连贯性检查

```typescript
// src/app/utils/ai/continuityChecker.ts
export function checkSceneContinuity(panels: StoryboardPanel[]): ContinuityIssue[] {
    const issues: ContinuityIssue[] = [];
    
    // 按场景分组
    const sceneGroups = new Map<string, StoryboardPanel[]>();
    panels.forEach(panel => {
        if (!sceneGroups.has(panel.sceneId)) {
            sceneGroups.set(panel.sceneId, []);
        }
        sceneGroups.get(panel.sceneId)!.push(panel);
    });
    
    // 检查每个场景
    sceneGroups.forEach((scenePanels, sceneId) => {
        // 1. 检查建立镜头
        const hasEstablishing = scenePanels.some(p => 
            p.shot === '远景' || p.shot === '全景'
        );
        if (!hasEstablishing && scenePanels.length > 2) {
            issues.push({
                type: 'missing_establishing',
                sceneId,
                message: '场景缺少建立镜头',
                severity: 'warning'
            });
        }
        
        // 2. 检查角色连贯性
        const allCharacters = new Set<string>();
        scenePanels.forEach(p => p.characters.forEach(c => allCharacters.add(c)));
        
        scenePanels.forEach((panel, index) => {
            if (index > 0) {
                const prevChars = new Set(scenePanels[index - 1].characters);
                const currChars = new Set(panel.characters);
                
                // 检查角色突然消失
                prevChars.forEach(char => {
                    if (!currChars.has(char) && allCharacters.has(char)) {
                        issues.push({
                            type: 'character_disappear',
                            panelNumber: panel.panelNumber,
                            message: `角色"${char}"突然消失`,
                            severity: 'info'
                        });
                    }
                });
            }
        });
        
        // 3. 检查时间连贯性
        const timeOfDays = scenePanels
            .map(p => p.scene?.timeOfDay)
            .filter(t => t);
        if (new Set(timeOfDays).size > 1) {
            issues.push({
                type: 'time_inconsistency',
                sceneId,
                message: '同一场景内时间不一致',
                severity: 'error'
            });
        }
    });
    
    return issues;
}
```
