# 项目库提示词生成优化实施方案

## 🎯 核心发现

经过全面检查，发现了一个**关键问题**：

### ❌ 缺少"AI智能生成提示词"功能

**现状**：
- 提示词只在"AI一键提取"时生成一次
- 用户手动添加资源后，提示词字段为空
- 用户修改描述后，提示词不会更新
- 只能手动编辑或使用"智能优化"（需要已有提示词）

**影响**：
- 用户体验差：不知道如何写专业提示词
- 功能不完整：无法充分利用AI能力
- 提示词质量不一致

**用户痛点场景**：
```
1. 用户手动添加角色"李四"
2. 填写描述："30岁男性，短发，戴眼镜，穿西装"
3. 想要生成图片，但 fullBodyPrompt 和 facePrompt 都是空的
4. 用户不知道如何写专业的AI绘画提示词
5. 只能放弃或手动编辑（质量无法保证）
```

---

## 💡 解决方案

### 方案：添加"AI智能生成"按钮

在每个提示词输入框旁边添加"AI生成"按钮，让用户可以基于当前描述一键生成专业提示词。

#### UI布局

**当前**：
```
[提示词输入框]
[智能优化] [复制]
```

**优化后**：
```
[提示词输入框]
[AI生成] [智能优化] [复制]
```

#### 功能对比

| 按钮 | 功能 | 使用场景 | 前提条件 |
|------|------|----------|----------|
| **AI生成** | 基于描述生成全新提示词 | 提示词为空或想重新生成 | 需要填写描述 |
| **智能优化** | 优化现有提示词 | 提示词已存在，想改进 | 需要已有提示词 |
| **复制** | 复制提示词到剪贴板 | 想在其他地方使用 | 需要已有提示词 |

### 1. 角色 (Character)

#### 1.1 修改文件
`src/app/components/asset-library/CharacterDetailView.tsx`

#### 1.2 添加生成函数

```typescript
const handleAIGenerate = (type: 'fullBody' | 'face') => {
    // 1. 检查描述
    if (!character.description && !character.appearance) {
        toast.error('请先填写角色描述或外貌特征');
        return;
    }
    
    // 2. 检查导演风格
    if (!project?.directorStyle) {
        toast.warning('未设定导演风格，将使用默认风格生成');
    }
    
    // 3. 生成提示词
    const engine = new PromptEngine(project?.directorStyle);
    let result;
    
    if (type === 'fullBody') {
        result = engine.forCharacterFullBody(character);
        onUpdate(character.id, { fullBodyPrompt: result.positive });
        toast.success('全身图提示词已生成！');
    } else {
        result = engine.forCharacterFace(character);
        onUpdate(character.id, { facePrompt: result.positive });
        toast.success('脸部图提示词已生成！');
    }
};
```

### 2. 场景 (Scene)

#### 2.1 修改文件
`src/app/components/asset-library/SceneTab.tsx`

#### 2.2 添加生成函数

```typescript
const handleAIGenerate = (type: 'wide' | 'medium' | 'closeup') => {
    // 1. 检查描述
    if (!scene.description && !scene.environment) {
        toast.error('请先填写场景描述或环境氛围');
        return;
    }
    
    // 2. 生成提示词
    const engine = new PromptEngine(project?.directorStyle);
    let result;
    
    switch (type) {
        case 'wide':
            result = engine.forSceneWide(scene);
            handleUpdateScene(scene.id, { widePrompt: result.positive });
            toast.success('远景提示词已生成！');
            break;
        case 'medium':
            result = engine.forSceneMedium(scene);
            handleUpdateScene(scene.id, { mediumPrompt: result.positive });
            toast.success('中景提示词已生成！');
            break;
        case 'closeup':
            result = engine.forSceneCloseup(scene);
            handleUpdateScene(scene.id, { closeupPrompt: result.positive });
            toast.success('特写提示词已生成！');
            break;
    }
};
```

#### 2.3 添加UI按钮

在每个提示词区域（远景/中景/特写）添加"AI生成"按钮。

### 4. 服饰 (Costume)

#### 4.1 修改文件
`src/app/components/asset-library/CostumeTab.tsx`

#### 4.2 添加生成函数

```typescript
const handleAIGenerate = () => {
    if (!costume.description) {
        toast.error('请先填写服饰描述');
        return;
    }
    
    const engine = new PromptEngine(project?.directorStyle);
    const character = assets.characters.find(c => c.id === costume.characterId);
    
    if (!character) {
        toast.warning('未关联角色，将生成通用服饰提示词');
    }
    
    const result = engine.forCostume(costume, character);
    handleUpdateCostume(costume.id, { aiPrompt: result.positive });
    toast.success('服饰提示词已生成！');
};
```

与道具类似，添加"AI生成"按钮。
