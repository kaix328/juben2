### ⚠️ 存在的问题

#### 🔴 严重问题（影响生成质量）

1. **AI 提取数据质量不稳定**
   - 位置：`assetExtractor.ts` 第 30-80 行
   - 问题：AI 返回的 JSON 格式不稳定，经常需要自适应归类
   - 影响：提取的角色/场景描述可能不完整或不准确

2. **提示词过于冗长**
   - 位置：`promptEngine.ts` 各 `for*` 方法
   - 问题：生成的提示词包含大量中文和重复信息
   - 影响：某些 AI 绘图平台（如 Stable Diffusion）对中文支持差，提示词过长会降低生成质量

3. **缺少提示词验证**
   - 位置：整个 `promptEngine.ts`
   - 问题：没有验证生成的提示词长度、格式、关键词密度
   - 影响：可能生成超长或无效的提示词

#### 🟡 重要问题（影响用户体验）

4. **字段映射不一致**
   - 位置：`assetExtractor.ts` 第 200-350 行
   - 问题：AI 提取的字段名（如 `hair`）与类型定义（`hairStyle`）不匹配
   - 影响：数据丢失或显示不正确

5. **提示词预览缺失**
   - 位置：`useImageGeneration.ts`
   - 问题：用户无法在生成前预览优化后的提示词
   - 影响：用户不知道实际发送给 AI 的内容

6. **批量生成无进度详情**
   - 位置：`useImageGeneration.ts` 第 280-340 行
   - 问题：批量生成时只显示总进度，不显示当前正在生成的资产名称
   - 影响：用户体验差，无法判断卡在哪里

#### 🟢 次要问题（可优化）

7. **硬编码的映射表**
   - 位置：`promptEngine.ts` 第 60-90 行
   - 问题：景别、角度、运动的映射表硬编码在代码中
   - 建议：应该使用统一的常量文件（已有 `cinematography.ts`）

8. **缺少平台适配**
   - 位置：`promptEngine.ts` 第 600-650 行
   - 问题：`formatForPlatform` 方法存在但未在实际生成中使用
   - 影响：无法针对不同平台（Midjourney/SD/Runway）优化提示词

9. **服饰与角色关联弱**
   - 位置：`promptEngine.ts` 第 380-420 行
   - 问题：服饰生成时虽然传入了角色，但关联信息利用不充分
   - 影响：生成的服饰图可能与角色外貌不一致

### 问题 2：提示词过于冗长

**现状示例：**
```
角色名, char_zhangsan_001, 男性，25岁，身高180cm，黑色短发，剑眉星目，身穿黑色长袍，佩戴玉佩, 全身正视图, full body shot, standing pose, front view, 白色背景, white background, simple background, 超高质量, 8K分辨率, 专业级, 精细细节, 完美构图, masterpiece, best quality, 8k, ultra detailed, professional, perfect composition, character design, reference sheet
```

**问题：**
- 中英文混杂，某些平台不支持中文
- 重复信息过多（如"高质量"重复多次）
- 总长度超过 200 tokens，影响生成效果

**改进方案：**
1. 根据平台选择语言（SD 用纯英文，国产平台可用中文）
2. 去重和精简，保留核心关键词
3. 限制总长度在 150 tokens 以内
4. 使用权重语法（如 `(keyword:1.2)`）替代重复

### 建议 3：创建提示词优化器

**文件：** `src/app/utils/promptOptimizer.ts`（已存在，需增强）

**新增功能：**
```typescript
export class PromptOptimizer {
  /**
   * 精简提示词（去重、压缩）
   */
  static simplify(prompt: string, maxLength: number = 150): string {
    // 1. 分词
    const parts = prompt.split(',').map(p => p.trim());
    
    // 2. 去重
    const unique = [...new Set(parts)];
    
    // 3. 按权重排序（保留最重要的）
    const sorted = this.sortByImportance(unique);
    
    // 4. 截断到最大长度
    return this.truncateToLength(sorted, maxLength);
  }
  
  /**
   * 转换为平台专用格式
   */
  static adaptToPlatform(prompt: string, platform: string): string {
    switch (platform) {
      case 'stable-diffusion':
        return this.toStableDiffusion(prompt);
      case 'midjourney':
        return this.toMidjourney(prompt);
      case 'runway':
        return this.toRunway(prompt);
      default:
        return prompt;
    }
  }
  
  /**
   * 验证提示词质量
   */
  static validate(prompt: string): ValidationResult {
    const issues: string[] = [];
    
    // 长度检查
    if (prompt.length > 500) {
      issues.push('提示词过长，建议精简到 150 tokens 以内');
    }
    
    // 关键词密度
    const keywords = this.extractKeywords(prompt);
    if (keywords.length < 3) {
      issues.push('关键词过少，建议增加描述性词汇');
    }
    
    // 中英文混杂检查
    const hasChinese = /[\u4e00-\u9fa5]/.test(prompt);
    const hasEnglish = /[a-zA-Z]/.test(prompt);
    if (hasChinese && hasEnglish) {
      issues.push('提示词中英文混杂，某些平台可能不支持');
    }
    
    return {
      isValid: issues.length === 0,
      issues,
      score: this.calculateScore(prompt)
    };
  }
}
```
