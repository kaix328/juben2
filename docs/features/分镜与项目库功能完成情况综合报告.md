## ⚠️ 发现的问题

### 项目库 - 提示词预览功能Bug

**问题描述**:
在 `useImageGeneration.ts` 中，当启用预览时，`handleGenerateCharacterFullBody` 等函数直接 `return`，但没有正确等待用户确认。

**影响**:
- 启用预览后，点击生成按钮不会弹出预览对话框
- 生成流程中断

**位置**:
```typescript
// src/app/pages/AssetLibrary/hooks/useImageGeneration.ts:241-248
if (enablePromptPreview) {
    setPromptPreview({
        type: 'character-full',
        id: characterId,
        originalPrompt: character.fullBodyPrompt,
        optimizedPrompt: result.positive,
        negativePrompt: result.negative,
    });
    return; // ❌ 直接返回，没有等待用户确认
}
```

**需要修复**: 所有生成函数（角色、场景、道具、服饰）都有相同问题
