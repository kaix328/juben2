## ⚠️ 发现的问题

### 🔴 严重问题（需立即修复）

#### 1. 类型不一致 ⚠️⚠️⚠️

**问题**:
```typescript
export interface StoryboardPanel {
    shot: string;              // ❌ 旧字段
    shotSize?: ShotSize;       // ❌ 新字段
    
    angle: string;             // ❌ 旧字段
    cameraAngle?: CameraAngleType; // ❌ 新字段
    
    cameraMovement?: string;   // ❌ 旧字段
    movementType?: CameraMovementType; // ❌ 新字段
}
```

**影响**: 
- 代码维护困难
- 容易出错
- 类型安全性降低

**解决方案**: 统一为专业类型 + 数据迁移工具

---

#### 2. 常量分散 ⚠️⚠️

**问题**: 景别/角度/运动映射在 5+ 个文件中重复定义

**文件**:
- `storyboardGenerator.ts` - SHOT_CODE_TO_CN
- `promptEngine.ts` - SHOT_TYPE_MAP
- `ShotCard.tsx` - shotOptions
- `qualityChecker.ts` - shotOrder
- ...

**解决方案**: 创建统一的 `constants/cinematography.ts`

---

#### 3. 缺少字段验证 ⚠️⚠️

**问题**: 没有运行时验证，可能导致无效数据

**解决方案**: 使用 Zod 添加验证层

---

### 🟡 中等问题（建议修复）

#### 4. 代码重复 ⚠️⚠️

**统计**:
- 提示词生成逻辑重复 > 5 处
- 映射逻辑重复 > 8 处
- 代码重复率约 30%

**解决方案**: 提取公共函数和辅助方法

---

#### 5. 错误处理不完善 ⚠️⚠️

**问题**:
```typescript
catch (error: any) {
    toast.error('分镜提取失败', {
        description: '请稍后重试',  // ❌ 通用提示
    });
}
```

**解决方案**: 创建自定义错误类，提供详细建议

---

#### 6. 性能问题 ⚠️

**问题**:
- Context 更新导致全局重渲染
- ListView 缺少虚拟滚动
- 100+ 分镜时出现卡顿

**解决方案**: 拆分 Context + 虚拟滚动
