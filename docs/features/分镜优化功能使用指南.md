# 分镜优化功能使用指南

## 🎉 新功能概览

已成功实现以下优化功能：

### ✅ 高优先级功能
1. **提示词验证器** - 检查提示词完整性和质量
2. **场景级连贯性检查** - 深度检查场景内的连贯性
3. **自动集成** - 在分镜生成时自动验证

### ✅ 中优先级功能
4. **提示词优化器** - 压缩和优化过长提示词

6. **批量优化工具** - 一站式批量优化

## 📚 功能详解

### 1. 提示词验证器

**文件**: `src/app/utils/promptValidator.ts`

**功能**：
- 检查提示词是否缺失
- 检查提示词长度是否合适
- 检查是否包含关键信息（角色、场景、景别）
- 检查视频必需字段（起始帧、结束帧）

**使用示例**：
```typescript
import { validatePrompt, validateAllPanels } from './utils/promptValidator';

// 验证单个分镜
const result = validatePrompt(panel, characters, scenes);
console.log(`评分: ${result.score}/100`);
console.log(`有效: ${result.valid}`);
console.log(`问题: ${result.issues.length}`);

// 批量验证
const batchResult = validateAllPanels(panels, characters, scenes);
console.log(`总体评分: ${batchResult.totalScore}/100`);
console.log(`有效: ${batchResult.validCount}, 无效: ${batchResult.invalidCount}`);
```

**输出示例**：
```
提示词质量评分: 85/100
有效分镜: 45, 需要优化: 5
问题统计: 错误 2, 警告 8, 提示 15

主要问题:
1. [ERROR] 缺少画面描述
2. [WARNING] 提示词过短（35字）
3. [WARNING] 缺少角色: 张三
```

### 3. 提示词优化器

**文件**: `src/app/utils/promptOptimizer.ts`

**功能**：
- 压缩过长提示词
- 按优先级保留关键信息
- 平台适配（可灵、Runway、Pika等）
- 语言转换（中文/英文/混合）

**使用示例**：
```typescript
import { optimizePrompt, optimizeAllPrompts, smartOptimize } from './utils/promptOptimizer';

// 优化单个提示词
const optimized = optimizePrompt(panel.aiPrompt, panel, {
    maxLength: 200,
    platform: 'kling',
    language: 'zh',
    prioritizeCharacters: true
});

// 批量优化
const optimizedPanels = optimizeAllPrompts(panels, {
    maxLength: 200,
    platform: 'kling'
});

// 智能优化（只优化过长的）
const smartOptimized = smartOptimize(panels);
```

**优化效果**：
```
[Prompt Optimizer] 优化完成: 23/50 个分镜被优化
[Prompt Optimizer] 总共节省: 1250 字符
```

### 4. 提示词模板系统

**文件**: `src/app/utils/promptTemplates.ts`

**预定义模板**：
- `standard` - 标准模板（通用）
- `characterFocus` - 角色特写
- `establishing` - 环境建立
- `action` - 动作场景
- `dialogue` - 对话场景
- `emotional` - 情绪氛围
- `technical` - 技术展示

**使用示例**：
```typescript
import { generateFromTemplate, batchGenerateFromTemplate, selectTemplateAuto } from './utils/promptTemplates';

// 使用指定模板
const { imagePrompt, videoPrompt } = generateFromTemplate(
    panel, 
    characters, 
    scenes, 
    'characterFocus'
);

// 自动选择模板
const templateName = selectTemplateAuto(panel);
console.log(`自动选择: ${templateName}`);

// 批量生成
const templatedPanels = batchGenerateFromTemplate(panels, characters, scenes);
```

**模板示例**：
```typescript
// 角色特写模板
imageTemplate: '{characterName}({characterTrigger}), {characterAppearance}, {shot}特写, {mood}表情, {lighting}光影'

// 生成结果
"张三(char_zhange09_f456), 30岁男性，短发，特写特写, 紧张表情, 低调光影"
```

### 5. 批量优化工具

**文件**: `src/app/utils/batchOptimizer.ts`

**功能**：
- 一站式批量优化
- 自动验证 + 优化 + 质量检查
- 自动修复常见问题
- 生成详细报告

**使用示例**：
```typescript
import { batchOptimizePanels, quickOptimize, deepOptimize } from './utils/batchOptimizer';

// 快速优化（推荐）
const result = await quickOptimize(panels, characters, scenes, directorStyle);

// 深度优化（包含模板重新生成）
const deepResult = await deepOptimize(panels, characters, scenes, directorStyle);

// 自定义配置
const customResult = await batchOptimizePanels(panels, characters, scenes, directorStyle, {
    validate: true,
    optimize: true,
    useTemplate: true,
    templateName: 'characterFocus',
    qualityCheck: true,
    autoFix: true,
    optimizeConfig: {
        maxLength: 200,
        platform: 'kling',
        language: 'zh'
    }
});

console.log(customResult.summary);
console.log(`成功: ${customResult.success}`);
console.log(`警告: ${customResult.warnings.join(', ')}`);
console.log(`错误: ${customResult.errors.join(', ')}`);
```

**输出示例**：
```
=== 批量优化摘要 ===
总分镜数: 50

【提示词验证】
- 评分: 85/100
- 有效: 45, 需优化: 5

【提示词优化】
- 优化数量: 23/50
- 原始长度: 12500 字符
- 优化后: 9800 字符
- 节省: 2700 字符 (21%)

【质量检查】
- 质量评分: 88/100
```

```typescript

import { quickOptimize } from '@/utils/batchOptimizer';

const handleOptimize = async () => {
    const result = await quickOptimize(panels, characters, scenes, directorStyle);

        // 更新分镜

};
```

## 🔧 配置选项

### OptimizeConfig（优化配置）

```typescript
{
    maxLength: 200,              // 最大长度
    platform: 'kling',           // 目标平台
    language: 'zh',              // 输出语言
    prioritizeCharacters: true,  // 优先保留角色
    prioritizeScene: true,       // 优先保留场景
    removeQualityTags: false     // 移除质量标签
}
```

### BatchOptimizeConfig（批量优化配置）

```typescript
{
    validate: true,              // 是否验证
    optimize: true,              // 是否优化
    useTemplate: false,          // 是否使用模板
    templateName: 'standard',    // 模板名称
    qualityCheck: true,          // 是否质量检查
    autoFix: true,               // 是否自动修复
    optimizeConfig: { ... }      // 优化配置
}
```

## 🐛 故障排除

### 问题1：提示词被过度压缩

**原因**：maxLength 设置过小

**解决**：
```typescript
optimizePrompt(prompt, panel, {
    maxLength: 250,  // 增加长度限制
    removeQualityTags: false  // 保留质量标签
});
```

### 问题2：验证评分过低

**原因**：缺少关键字段

**解决**：
```typescript
// 使用自动修复
const result = await batchOptimizePanels(panels, characters, scenes, directorStyle, {
    autoFix: true  // 启用自动修复
});
```

### 问题3：模板生成结果不理想

**原因**：变量缺失

**解决**：
```typescript
// 检查变量
const variables = extractVariables(panel, characters, scenes);
console.log('可用变量:', variables);

// 使用自动选择
const templateName = selectTemplateAuto(panel);
```
