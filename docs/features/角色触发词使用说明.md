# 角色触发词功能说明

## 什么是角色触发词？

角色触发词（Trigger Word）是一个唯一标识符，用于确保 AI 视频生成时角色外观保持一致。

### 格式示例
```
张三 → char_zhange09_f456
李四 → char_li6db_w012
王小明 → char_wangc0f6_y789
John Smith → char_johnsmit_m012
```

## 在分镜提示词中的显示

### 视频提示词
```
中景，张三(char_zhange09_f456): 30岁男性，短发，李四(char_li6db_w012): 25岁女性，长发，办公室场景...
```

### 图片提示词
```
张三(char_zhange09_f456), 30岁男性，短发，黑色西装，办公室，中景，平视...
```

## 自动生成时机

触发词会在以下情况自动生成：

1. ✅ **新建角色时** - 创建角色时自动生成
2. ✅ **修改角色名称时** - 名称改变时自动更新触发词
3. ✅ **AI 提取角色时** - 从剧本提取角色时自动生成
4. ✅ **同步导演风格时** - 批量为旧角色生成触发词

## 触发词的作用

1. **角色一致性** - 确保同一角色在不同分镜中外观一致
2. **唯一标识** - 每个角色有独特的触发词，避免混淆
3. **模型训练** - 可用于 LoRA 模型训练和 IP-Adapter

## 常见问题

### Q: 为什么需要触发词？
A: AI 视频生成模型需要一个稳定的标识符来识别角色，确保同一角色在不同镜头中保持外观一致。

### Q: 可以手动修改触发词吗？
A: 可以。在角色详情页可以手动编辑触发词，系统不会覆盖手动设置的值。

### Q: 旧项目的角色没有触发词怎么办？
A: 使用"同步导演风格"功能，系统会自动为所有角色生成触发词。

### Q: 触发词会影响生成效果吗？
A: 触发词主要用于角色识别，不会影响画面质量。它帮助 AI 理解"这是同一个角色"。

## 技术细节

### 生成规则

**中文名称**:
- 姓氏转换为拼音（如：张→zhang）
- 名字转换为 Unicode 编码
- 格式：`char_[姓氏拼音][名字编码]_[4位hash]`

**英文名称**:
- 直接提取字母和数字
- 格式：`char_[英文名]_[4位hash]`

### 支持的姓氏
张、王、李、刘、陈、杨、黄、赵、周、吴、徐、孙、马、朱、胡、郭、何、林、罗、高等常见姓氏。
