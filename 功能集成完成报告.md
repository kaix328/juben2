```typescript
// useStoryboardActions.ts
const handleBatchGenerateVideoPrompts = useCallback(async (
  selectedIds: Set<string>,
  platform: VideoPlatform = 'runway',
  onProgress?: (current: number, total: number) => void,
  onComplete?: () => void
) => {
  // 创建增强的 PromptEngine
  const engine = new PromptEngine(project.directorStyle, {
    useProfessionalSkills: true,
    targetPlatform: platform,
    includeNegative: false,
    qualityTags: 'professional',
  });

  // 批量生成视频提示词（支持上下文感知）
  const updatedPanels = storyboard.panels.map((panel, index) => {
    if (!selectedIds.has(panel.id)) return panel;
    
    const prevPanel = index > 0 ? storyboard.panels[index - 1] : undefined;
    const result = engine.forStoryboardVideo(panel, assets.characters, assets.scenes, prevPanel);
    
    return { ...panel, aiVideoPrompt: result.positive };
  });

  await onUpdateStoryboard({ ...storyboard, panels: updatedPanels });
}, [storyboard, assets, project, onUpdateStoryboard]);
```

## 🚀 使用方法

### 1. 批量导出提示词

**步骤**:
1. 在分镜编辑器中，点击底部批量操作栏的"导出提示词"按钮（绿色）
2. 在弹出的对话框中选择目标平台（Midjourney/Runway/Pika等）
3. 配置导出选项（是否包含角色定义、负面提示词等）
4. 点击"导出 Markdown"按钮
5. 文件自动下载为 `.md` 格式

**导出内容示例**:
```markdown
# 分镜提示词导出
# 平台: midjourney
# 导出时间: 2026-01-25T10:30:00.000Z
# 总分镜数: 10

## 负面提示词 (Negative Prompt)
low quality, worst quality, blurry...

## 角色定义 (Character Definitions)
=== 张三 ===
[Trigger Word] char_zhang_001
[Name] 张三
[Appearance] 30岁男性，黑色短发...

## 分镜提示词

### 分镜 1 (5秒)
**AI绘画提示词:**
中景镜头, medium shot, 张三(char_zhang_001)...
--ar 16:9 --style raw --v 6.1

**AI视频提示词:**
dolly in, camera moving forward, 5 seconds...
--resolution 1280x768 --duration 4s
```

---

### 2. 批量生成视频提示词

**步骤**:
1. （可选）选中需要生成的分镜，或不选则生成全部
2. 点击底部批量操作栏的"生成视频提示词"按钮（粉色）
3. 在弹出的对话框中选择目标平台（Runway/Pika/Kling/Sora）
4. 查看平台特性说明
5. 点击"开始生成"按钮
6. 等待生成完成（显示进度）
7. 生成的视频提示词自动保存到每个分镜的 `aiVideoPrompt` 字段

**生成的视频提示词示例**:

**Runway Gen-3 格式**:
```
medium shot, eye level, dolly in, camera moving forward, 5 seconds,
50mm lens, f/2.8, shallow depth of field,
张三(char_zhang_001) walking towards window, sunlight on face,
modern office, bright and spacious, floor-to-ceiling windows,
golden hour lighting, warm orange glow,
cinematic lighting, professional cinematography,
smooth motion, fluid animation
--resolution 1280x768 --duration 4s
```

**可灵格式**:
```
中景镜头，平视角度，推镜头，5秒，
张三(char_zhang_001)缓缓走向窗边，阳光洒在脸上，
现代化办公室，明亮宽敞，落地窗，
黄金时刻光线，温暖橙色，
电影感，专业摄影，高质量
#视频生成 #电影感
```

## 🎓 技术实现

### 核心技术栈
- **PromptEngine v2.0**: 统一提示词生成引擎
- **React Hooks**: 状态管理和副作用处理
- **TypeScript**: 类型安全
- **Sonner**: Toast 通知
- **Shadcn/ui**: UI 组件库

### 架构设计
```
StoryboardEditor (主组件)
├── BatchActionBar (批量操作栏)
│   ├── onExportPrompts → ExportPromptsDialog
│   └── onBatchGenerateVideoPrompts → VideoPromptGenerationDialog
├── useStoryboardActions (操作 Hook)
│   ├── handleBatchGenerateVideoPrompts (批量生成视频提示词)
│   └── handleExportPrompts (导出提示词)
└── PromptEngine (提示词引擎)
    ├── forStoryboardVideo (生成视频提示词)
    ├── forStoryboardImage (生成图像提示词)
    └── formatForPlatform (平台格式化)
```
