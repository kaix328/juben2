# 分镜功能改进 - 依赖清单

## 📦 新增依赖

### 运行时依赖

#### 1. zod (^3.22.4)
**用途**: 运行时数据验证

**使用位置**:
- `src/app/utils/validation/storyboardValidation.ts`

**功能**:
- 分镜面板数据验证
- 自动类型推导
- 详细的错误信息

**安装命令**:
```bash
npm install zod
```

---

#### 2. @tanstack/react-virtual (^3.0.0)
**用途**: 虚拟滚动优化

**使用位置**:
- `src/app/pages/StoryboardEditor/components/ListViewOptimized.tsx`

**功能**:
- 高性能列表渲染
- 支持 500+ 分镜
- 动态高度测量

**安装命令**:
```bash
npm install @tanstack/react-virtual
```

## ⚠️ 注意事项

### 1. 版本兼容性
- `zod` 需要 TypeScript 4.5+
- `@tanstack/react-virtual` 需要 React 16.8+

### 2. 打包大小
- `zod`: ~13KB (gzipped)
- `@tanstack/react-virtual`: ~8KB (gzipped)
- **总增加**: ~21KB

### 3. 浏览器兼容性
- 支持所有现代浏览器
- IE11 需要 polyfill

## 🚀 使用示例

### Zod 验证
```typescript
import { z } from 'zod';

const schema = z.object({
    id: z.string(),
    name: z.string().min(1),
});

const result = schema.safeParse(data);
if (result.success) {
    console.log('验证通过', result.data);
}
```

### React Virtual
```typescript
import { useVirtualizer } from '@tanstack/react-virtual';

const virtualizer = useVirtualizer({
    count: items.length,
    getScrollElement: () => parentRef.current,
    estimateSize: () => 400,
});
```
