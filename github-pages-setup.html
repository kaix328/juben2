🚀 GitHub Pages 配置助手

自动引导您完成 GitHub Pages 配置

1 点击下方按钮
将自动打开 GitHub Pages 配置页面
2 选择部署源
在 "Source" 下拉菜单中选择 "GitHub Actions"
3 保存设置
点击 "Save" 按钮完成配置
将在 5 秒后自动打开配置页面...
✅ 配置页面已打开！请在新标签页中完成配置。
