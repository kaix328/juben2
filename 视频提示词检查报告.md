## 🎬 核心架构

### PromptEngine 类（统一引擎）

```typescript
class PromptEngine {
  // 核心方法
  forStoryboardVideo()    // 生成视频提示词
  forStoryboardImage()    // 生成图片提示词
  forCharacterFullBody()  // 生成角色全身图
  forCharacterFace()      // 生成角色脸部图
  forSceneWide()          // 生成场景远景
  forProp()               // 生成道具图
  forCostume()            // 生成服饰图
  
  // 静态方法
  generateTriggerWord()   // 生成触发词
  formatForPlatform()     // 平台格式化
  getPlatformConfig()     // 获取平台配置
}
```

### 专业技能库

1. **cinematography.ts** - 电影摄影常量
   - 13 种景别类型
   - 7 种镜头角度
   - 19 种镜头运动

2. **colorGrading.ts** - 调色预设库
   - 🎬 电影风格（5种）
   - 🌅 时段调色（5种）
   - 🎭 情绪氛围（6种）
   - 🎥 类型片风格（4种）
   - 🌏 地域风格（5种）

3. **promptValidator.ts** - 提示词验证
   - 格式验证
   - 质量检查
   - 去重优化
   - Token 限制
